Uma nova "cara" para o Orientek

Voltado para a organização de competições do esporte orientação, o Orientek é um sistema desenvolvido pelo Programa Azimute norte desde 2017. No entanto o avanço das tecnologias implementadas pelos maiores navegadores web viu-se a possibilidade de implantação de funcionalidades offline. Junto com isso o sistema também passa por uma redefinição de sua interface.

A nova interface segue o padrão de desingn material que possui uma série de regras e ferramentas que simplificam e agilizam o processo de produção. Usando o framework materialize combinado com o Angular o site passou a implementar uma Single Page Application alimentada por uma API implementada em PHP.
