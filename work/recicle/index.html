Mais uma grande ferramenta rumo a um mundo mais limpo.

Recile é um aplicativo proposto pela ONG Lixo e Cidadania em 2018 que foi desenvolvido como trabalho de conclusão de curso durante o mesmo ano. Para isso o aplicativo conta com um cliente em dispositivos Android e um cliente web para a adiministração do sistema.

O módulo Android foi desenvolvido com o framework padrão do próprio sistema operacional otimizando a performance. Além disso a interface segue ao padrão material é oferecida nativamente pelo sistema. Já o módulo web possui uma interface parecida que foi implentada graças ao framework Materialize.

Já no inicio do trabalho notou-se que a necessidade em um serviço de backend complexo demais para um aluno técnico seria necessário. Então optou-se pelo uso da plataforma Firebase, disponibilizada gratuitamente, para a implementação deste trabalho. Com o uso desta plataforma o toda a infraestrutura de backend, autenticação e banco de dados fica a cargo da plataforma.
